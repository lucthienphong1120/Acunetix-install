## Docker setup guice

### Build image

```sh
docker build -t acunetix_13 .
```

### Run the container

```sh
docker run -d -p 3444:3443 --name acunetix acunetix_13
```

## Local setup guide

Acunetix 13 Linux Installï¼š
1. Install using `bash acunetix_13.0.200217097_x64.sh`
